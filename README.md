# je = json editor

json editor port to npm

[![npm version](https://badge.fury.io/js/je.svg)](http://badge.fury.io/js/je)

## Install 

    [sudo] npm install -g je
  
## Usage

in terminal 

```
  je
```
  
or
 
```
  je -f xxx.json
  je --url http://xxxx
```
